# About Ike Knowledge Base

The full-cycle knowledge base: this set's change set alone, assembled and classified by ike:kb-assemble and installed into the data-source directory for verification in Komet. Ledger-only by doctrine (IKE-Network/ike-issues#909, #921): the Tinkar starter data was a one-time input whose value is captured in the ledger's declarations, so the store is single-version by construction — every chronology at the inception stamp pair, exactly what LedgerGatesIT's whole-store stamp gate pins. Verification edits are captured under the installed store's changeSets directory and feed the next iteration as CHANGESET file inputs below.
